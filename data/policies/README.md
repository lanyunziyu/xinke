# 政策数据目录

此目录用于存放购房相关的政策文档。

## 文件组织结构

建议按以下方式组织政策文档：

```
policies/
├── beijing/
│   ├── 限购政策.md
│   ├── 贷款政策.md
│   ├── 公积金政策.md
│   └── districts/
│       ├── 朝阳区.md
│       ├── 海淀区.md
│       └── ...
├── national/
│   ├── 贷款利率.md
│   └── 税费政策.md
└── examples/
    └── 政策文档模板.md
```

## 政策文档格式

每个政策文档应包含：

1. 政策名称
2. 生效日期
3. 适用范围
4. 具体规定
5. 相关链接（如果有）
